MCA Pop-up Directive (Documentation)
Type :
Service
Usage
Showing:
mcaPopUp.show({
text:"Partner Funder is SuccessFully Added!",
clickOutSideClose: true,
transparency:0.6,
buttonText:"Ok",
animation:true
});
Closing:
mcaPopUp.close();
Attribute Name Type Default Value Desc
text Required - Text content to be displayed on Pop-Up.
clickOutSideClose optional true If true, pop will closed even you click on dim area
If false, Pop-up close is restricted to button.
transparency optional 0.5 Amount of dim effect in background (0.0 - 1.0).
buttonText optional Close Text to be displayed on close button of pop-up.
animation optional true animation effect during pop-up.
